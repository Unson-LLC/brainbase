# Brand Assets Repository
ロゴ等の正本を一元管理する場所。編集はここで行い、各プロダクト/資料では参照のみ。

## 構成
- `baao/`：BAAOロゴ（SVG/PNG）、反転版、最小サイズ・クリアスペースメモ
- `unson/`：UNSONロゴ
- `techknight/aitle/`：Aitleロゴ
- `salestailor/`：SalesTailorロゴ

## 運用ルール
- 正本はSVG。配布用にPNG/WebPを書き出す。
- 変更時はバージョンをディレクトリ内 README に記載し、ブランドガイドから参照パスを更新。
- クリアスペースと最小使用サイズを各ディレクトリREADMEに明記。
