# Custom Commands

このディレクトリ配下の `*.md` は、Claude Code の Custom Command として利用します。
（Codex CLI でも `AGENTS.md` のルールに従えば同じコマンド定義を参照できます）

## Commands

| Command | Doc | Purpose |
|---|---|---|
| `/commit` | `.claude/commands/commit.md` | 変更を標準フォーマットでコミット |
| `/create-pr` | `.claude/commands/create-pr.md` | PR作成 |
| `/merge` | `.claude/commands/merge.md` | PR作成〜マージ |
| `/handover` | `.claude/commands/handover.md` | セッション引き継ぎ（`HANDOVER.md` 生成） |
