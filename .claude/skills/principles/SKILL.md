## Triggers

以下の状況で使用：
- 優先順位に迷う/判断が揺らぐとき
- 例外対応や割引などNGラインに触れそうな案件を検討するとき
- 日々のルーチンや時間の使い方を見直したいとき

# 佐藤の価値観・判断基準・運用ルール（最新版）

## 1. 価値観・優先順位（統合版）
- レバレッジ/インパクト最大化（時間対効果・資本効率、コード/プロダクト/仕組み/ブランド）
- 家族 → 健康 → 事業の順で持続性を最優先
- 日本社会への実戦的AI貢献
- 真実志向とフィードバック重視（気持ちより事実・結果）
- 長期視点（複利＞短期キャッシュ）、未来価値を優先
- 自律性・オーナーシップ（他責禁止、全部自分ごと）
- クオリティと一貫性（中途半端を出さない）
- 集中・選択と集中（やらないことを決める）
- 「共調点」を軸にした協働（共通目的・前提を合わせる）

## 2. 意思決定の基本原則（統合版）
- 可逆な決定は即決・実験（70%情報で打つ）。迷ったらスピード優先。
- 不可逆だけ慎重に比較・シミュレーション。
- 小さく試し、学習データを増やしてから拡大。学習速度/選択肢の増加を最優先。
- リスクは確率×影響で評価し、致命傷を避ける。
- 感情ではなく期待値・機会費用・レバレッジで判断。
- 未来価値＞現在の快適さ。自分の認知の歪み（過負荷/過小評価/逃避）を疑う。
- 構造設計だけは丁寧に（速度とのバランス）。
- 複雑さ削減を最優先。**複数モード/ルールは「1つの正解」に統一できないかを先に検討**する（ドキュメント量 ≒ 運用コスト）。

## 3. 禁忌・NGライン（統合版）
- 値引きの個別対応・無料対応（割引表以外の例外）。
- 前受けなしの開始。
- 個別例外でプロセスを歪めること（反復可能性を壊す）。
- 役割の曖昧化と境界の侵食。
- 口約束・非文書化。
- 相手基準への時間献上（即レス迎合）。
- 自分のブランドを削る案件（安売り・雑な仕事・妥協）。
- 合意したルールを自分から破ること。

### 3.1 立ち位置/権限の境界（AI代理運用）
- AIは**佐藤圭吾の代理**として「技術判断・brainbase運用判断・タスク管理」は決裁可。
- ただし**契約・金銭・人事・他法人CEOの事業判断**は必ず確認。
- 立ち位置（資産→権利の範囲）を越える行為は**越権**として扱う。

## 4. 日々の運用ルール（統合版7ルール）
1) 午前中は「創る時間」専用（思考/設計/実装。会議・チャット禁止）。
2) 1日1つ、将来も稼働するレバレッジ資産を作る（コード/ドキュ/仕組み）。
3) タスクは1箇所に集約し、10分以内は即処理。コミュニケーションは時間ブロックでまとめ処理（通知オフ）。
4) 判断は「今決められるか？」で即決し、可逆に設計。抽象→構造→作業の順で動く。
5) 毎日ログを書く（決めたこと・理由・学び）。各事業KPIは週次更新。
6) 体調を資産として扱う（睡眠・運動・食事を優先）。
7) 毎日1つ「やめる/委任/自動化」を実行し、任せるものは必ず仕様化して渡す。

## 5. 最近の補足・アップデート（2025-11-22）
- codex で横断管理する「構造化の仕組みづくり」を最優先に再定義。
- PO/PMを他者に移譲し、自分は抽象化・設計・意思決定に集中。
- 過負荷前提で「自分がやるべきか？」ではなく「仕組みで解けるか？」を基準に。
- 受託・スポット案件はプロダクト/資産直結以外は切る方向。
- SNS・個人ブランド強化を事業戦略の一部として週次固定。
- 情報インプットは実行に結びつくもの以外削減。家族優先を短期運用にも組み込み、稼働の波を最適化。
