# X 海外キュレーション戦略

## Triggers

- 海外の最新情報を日本向けに発信したいとき
- 情報鮮度で先行者優位を取りたいとき
- `/curate` と呼ばれたとき

---

## 戦略の本質

**海外バズ × ユーザー需要 × 軸足 = 高価値キュレーション**

```
海外でバズってる情報（鮮度）
     ↓
日本ではまだ知られていない（希少性）
     ↓
自分の視点・実体験を加える（軸足）
     ↓
「海外で話題の〇〇、試してみた」
```

**先行者優位**: 同じ情報でも最初に発信した人が認知を取る

---

## 4ステップ

### Step 1: 海外バズ発見

**実行**:
```bash
python /Users/ksato/workspace/shared/_codex/common/ops/scripts/x_research/buzz_discovery.py \
  --query "Claude Code lang:en" \
  --min-engagement 500
```

**選定基準**:
| 項目 | 基準 | 優先度 |
|------|------|--------|
| **フォロワー適合** | 反応パターンに合致（Step 2参照） | ★★★ 最重要 |
| 鮮度 | 48時間以内（理想は24時間） | ★★☆ |
| Engagement | 50+（Viewsより実際の反応を重視） | ★★☆ |
| 内容 | 日本で未紹介 or 深掘り価値あり | ★☆☆ |

**優先度高**（フォロワー視点で選定）:
- ⚠️ **AIの限界・落とし穴**（「これはできない」「ここが危険」）
- 🔄 **ツール比較**（実践者目線で「こっち選んだ理由」）
- 💥 **期待vs現実**（「〇〇できると思ってた→詰んだ」）
- 🛠️ **失敗→解決の実践談**

**優先度低**（避けるべき）:
- 純粋な技術Tips（コード例のみ）
- 新機能紹介のみ（実践なし）
- 数字・実績報告

---

### Step 2: フォロワー需要確認（実データ基準）

**チェックリスト**:
- [ ] 日本語で同じ情報が既に出回っていないか
- [ ] **フォロワーが反応するパターンに合致するか**（下記参照）
- [ ] 解説・翻訳する価値があるか

**フォロワーが反応するパターン**（実データ分析: 2025-12）:

| パターン | 反応率 | 例 |
|---------|-------|-----|
| **警鐘・問題提起** | ★★★ | 「これ危険」「やめた方がいい」 |
| **期待vs現実** | ★★★ | 「〇〇できると思ってた→できなかった」 |
| **ツール比較（実践者目線）** | ★★☆ | 「〇〇 vs △△、こっち選んだ理由」 |
| **失敗談→解決策** | ★★☆ | 「ハマった→こう解決した」 |

**反応しないパターン**（避けるべき）:

| パターン | 理由 |
|---------|------|
| 純粋な技術Tips | 詳細すぎてターゲット外 |
| 数字報告のみ | 自慢に見える |
| ツール紹介のみ | 比較・実践がないと刺さらない |
| 細かいバグ修正話 | ニッチすぎる |

**フォロワー層の属性**:
- 経営者・1人社長（技術者ではない）
- AI活用に興味あるが、実装は自分でやらない
- 「AIで何ができるか」より「AIで何が危険か」に関心

---

### Step 3: 軸足ベースで翻訳+実体験

**1行目テンプレート**:
```
速報系: 「海外で話題の〇〇」「これ、日本ではまだ知られてないけど」
実践系: 「海外で話題の〇〇、試してみた」「英語圏でバズってる〇〇を実践」
解説系: 「〇〇（開発者）が公開した〇〇を解説」
```

**構成**:
```
[1行目: 情報の希少性・鮮度を示す]
[空行]
[元情報の要約（翻訳）]
[空行]
[自分の実体験・軸足ベースの解説]
[空行]
[結論・日本ユーザーへの示唆]
[空行]
[元ツイートURL or スレッド]
```

**軸足チェック**（佐藤圭吾の場合）:
- 7事業を1人で並行運営
- brainbase（AI管理OS）開発者
- Claude Code Meetup登壇者
- 「AIにマネジメントを丸投げする男」

---

### Step 4: 簡易レビュー（3項目）

| チェック | 基準 |
|---------|------|
| 情報鮮度 | 24-48時間以内か、日本初出か |
| 翻訳精度 | 元情報を正確に伝えているか |
| 軸足価値 | 単なる翻訳でなく自分の視点があるか |

**NG例**:
- 既に日本で広まっている情報
- 単なる翻訳（価値上乗せなし）
- 元情報の誤訳・誤解
