## Triggers

以下の状況で使用：
- 知名度はあるのに購入されない原因を切り分けたいとき
- 広告媒体選定やメッセージが適切か失敗パターンでチェックしたいとき
- イノベーター普及/ダブルイノベーターの観点で導入戦略を練りたいとき

本で述べられている本質的な内容は、主に**失敗のパターンを分析し、それを回避または活用するためのマーケティング理論と戦略**に焦点を当てています。

以下に、ソースに基づいた本書の主要な概念と構造を詳細にまとめます。

## 本書の根幹をなす概念と目標

本書は、**紙書籍『失敗から学ぶマーケティング 売れないモノには理由がある』**（ISBN978-4-297-12459-5）を元に制作された電子書籍です。

*   **目的:** 失敗の事例を分析することで、**失敗にはパターンがある**ことを示し、そのパターンから学び、マーケティング理論から逸脱した失敗を回避し、成功確率を上げるための対抗方法を学ぶことです。
*   **対象読者（推奨される人）:**
    *   これからマーケティングを学ぼうとする学生。
    *   他部署から商品開発、広告宣伝、新サイトの企画を任された人。
    *   なかなかヒットが生まれず悩んでいる担当者。
    *   ヒット商品には関係ないが、知的好奇心の強い人。

## I. マーケティングにおける失敗の構造と指標

### 1. U&E（知る/理解/好意/トライアル/レギュラー）モデルと課題

U&E（Usage & Exposure: 使用と露出）モデルは、**知名**、**理解**、**好意**、**トライアル**、**レギュラー**の5つの指標から構成されています。

| 指標 | 概要 | 課題・失敗の例 |
| :--- | :--- | :--- |
| **知名** | 商品名を知っていること。「知られなければ『ない』のと同じ」。 | **初期開発でも知名度を利用できない**。知名度向上を目的とした広告媒体の選択ミス（パターン1/4）。 |
| **理解** | 商品のメリットやベネフィットを理解していること。 | **理解度が上がらない主な原因**として、企業が「広告はイメージが最も重要」と考え、イメージだけの広告に注力すること（パターン1）や、インパクト重視の広告を打つこと（パターン2）がある。 |
| **好意** | 商品に対して好意的な感情を抱くこと。 | **理解されても好意/購入意向に結びつかない**ケースがある（例: 日経新聞の「良質な記事」）。 |
| **トライアル** | 試用または購入すること。 | - |
| **レギュラー** | 定期的な購入者となること。 | - |

**コンバージョン・レートとリテンション・レート:**
*   **コンバージョン・レート（トライアル÷知名度）**：**広告の問題**（コミュニケーション・コンバージョン・レート：理解度÷知名度）をチェックする補助指標。
*   **リテンション・レート（レギュラー÷トライアル）**：**商品コンセプトの問題**（コミュニケーション・リテンション・レート：好意度÷理解度）をチェックする補助指標。

### 2. 広告とメディア選択の課題

*   **広告費と知名度の関係（S字曲線）:** 広告費に対する知名度の向上は、最初ゆっくり上がり、40%から60%の間で急激に上昇し、その後再び緩やかになるという傾向があります。
*   **ターゲットとなる視聴者の質の問題:** テレビ広告視聴者は、ネット動画視聴者に比べて、**興味度/購入可能性**が高い傾向にあります。
*   **知名度を上げる目的で使ってはいけない媒体の分類（パターン1～4）**：
    *   **パターン1（限定的なケースで知名度向上に有効）:** ネット媒体、ティッシュやパンフレット配布。
    *   **パターン2（理解度に有効）:** ネット媒体。
    *   **パターン3（トライアルに有効）:** POP、ネット通販。
    *   **パターン4（コントロール不可能）:** クチコミ。

## II. 消費者行動とイノベーター理論

### 1. イノベーター理論とダブルイノベーター理論

イノベーター理論は、社会に新商品が普及するプロセスを、消費者を**イノベーター（約2.5%）**、**アーリーアダプター（約13.5%）**、**フォロワー（約80%）**に分類して説明します。

*   **イノベーターの役割:** イノベーターは「ヒット商品の導火線」であり、**自力で探して買ってくれる**、**ピンとくればすぐ買う**という特徴を持ち、「知識」「差別分析」「好奇心」を共通項として持っています。
*   **イノベーターの誤解:** イノベーター像として、「女子高生」「最先端を行く人々」「流行を追いかける飽きっぽい人たち」「有名人や芸能人」が挙げられるが、これらはしばしば誤解されている要素です。
*   **ダブルイノベーター理論:** 市場に2種類のイノベーターが存在するという考え方です。
    *   **伝統型イノベーター:** **スペック重視**。
    *   **革新型イノベーター:** **ベネフィット重視**（ゲームのルールを変える）。

**革新型イノベーターが発生しやすい3つの条件**:
1.  **「生活者のルール/マナー/方向性」が強固な業界や産業**（例: 写真はきれいに撮るべき）。
2.  **成熟期にある業界や産業**。
3.  **規格よりベネフィット志向**（例: ベネフィットに反応する商品には興味があるが、アーリーアダプターが反応する規格には興味がない）。

### 2. 価値観と行動

*   **デモグラフィック分類の限界:** 従来のマーケティングで使われてきた年齢や性別などの人口統計学的（デモグラフィック）な分類は、ターゲット設定として機能しない場合があります。
*   **価値観ヒエラルキー:** アメリカの心理学者ヤンケロビッチ博士が提唱したもので、価値観は階層構造を持ち、**価値観**が決まれば**好みや行動**も決定されるというものです。
    1.  Source (ソース): 持って生まれた性格。
    2.  Value (ヴァリュー): 価値観（基本ニーズ）。
    3.  Criteria (クライテリア): 生活基準（基本ニーズ）。
    4.  Taste (テイスト): 生活の志向、好み、感性（直接ニーズ）。
    5.  Manufestation (マニフェステーション): 生活行動（消費行動）。
*   **価値観ターゲット:** 価値観をベースに、業界別の価値観グループ（例: 「健康」に関する自助努力型/他力本願型）を作ることが、マーケティングの最適解の一つです。

### 3. 認知ベースの行動と対抗手段

*   **認知ベースの行動モデル:** 人は必ずしも事実に基づいて行動するのではなく、**「認知ベースの行動」**、つまり**「認知・信念・思い込み」**に基づいて行動することがあります。
    *   **基礎モデル:** 事実 → 情報 → 経験 → 事実 → 知識。
    *   **学習反復モデル:** 情報 (知識) → 経験 → 事実 → 知識。
    *   **知識モデル:** 情報 (知識) → 知識。
    *   **確信した情報モデル:** 情報 (知識) → 事実。
*   **不利な状況に追い込まれた時の3つの対抗手段**:
    1.  **正面反撃戦略:** 競合の攻撃に対し、真正面から反撃する戦略。攻撃の規模が大きい場合に有効。
    2.  **空手内受け反撃戦略:** 攻撃の方向から角度をつけ、生活者の関心をそらす戦略。
    3.  **合気道一教反撃戦略:** 攻撃の力を利用し、結果的に自社の有利な方向へ導く戦略。

## III. 商品とブランドの戦略

### 1. プロダクトライフサイクル理論と戦略ミス

プロダクトライフサイクルは、個々の商品やブランドではなく**「商品ジャンル」**で捉えるべきです。

*   **導入期:** 旧ジャンルとの**差別化**のある商品が成功。
*   **成長期:** 現ジャンルでの**優位性**のある商品が成功。**ベネフィット訴求**に切り替えることが重要。
*   **成熟期:** 現ジャンルとの**差別化**のある商品が成功。

**成長期の戦略3点セット（成功したアップル製品の例）**:
1.  「差別化せず」に**優位性**を向上させる。
2.  「需要が大きいベネフィット」を訴求する。
3.  「普及と価格帯の商品」を用意する。

### 2. ブランドコンセプトの失敗（コンセプトの鈍化）

ブランドが生活者に対して持つ**「約束」**が崩れると失敗に繋がります。

*   **「乱発＆ボケ」症状:** 親ブランドの「約束」から乖離したエクステンションを次々と市場投入し、ブランドイメージが曖昧になること。
    *   **第1の治療法:** 親ブランドの約束を明確にし、そこから離れたエクステンションを切り離す。
    *   **予防策:** 肝となる生活者の約束を必ず残すよう社内で徹底する。
*   **「形だけのお付き合い」症状:** ブランドの「約束」とは別の、表面的な要素で商品展開をしてしまうこと（例: カゴメがトマトだけでなく缶コーヒーやサイダーを出す）。

### 3. ブランドイメージの維持戦略

ブランドを維持する戦略には2つの型があります。
*   **記号性継続型（マールボロ）:** ブランドイメージの**本質**を維持し、時代を超越する。
*   **意味性継続型（コカコーラ）:** 時代の流れに合わせて、ブランドイメージを構成する**記号（表現要素）**を変化させる。

### 4. 意識のフリッジ理論（パンツ理論）

企業がブランドを開発する際、**企業からは1本の道に見える（「らしく」ある）**にもかかわらず、**生活者からは2本の道に見える**現象があり、これが**意識のフリッジ（ズレ/ギャップ）**を引き起こし失敗の原因となります。

*   **「らしくないこと」の失敗:** ブランドの約束が強ければ強いほど、「らしくない」行動（イメージ、価格、デザイン、相容れない商品）は失敗につながる可能性が高くなります。
*   **認知地図の活用:** 生活者のブランドに対するイメージの**認知地図**を活用し、競合の「陣地」を攻める戦略（例: アップルが「スマート」陣地から「犬バカ」陣地へ攻める）を立てることができます。

## IV. 競合・市場規模・劣位戦略

### 1. ガリバーと弱者（クープマンの目標値）

*   **クープマンの目標値:** 市場シェアにおいて、**独占シェア（73.9%）**、**安定シェア（41.7%）**、**影響シェア（26.1%）**、**認知シェア（10.9%）**、**存在シェア（6.8%）**の5つの指標が存在します。
*   **ガリバーと弱者の戦い方:**
    *   トップ企業（ガリバー）は、シェア41.7%を超えると強固な**「守りの力」**を持ちます。
    *   弱者（ガリバーより小さい企業）は、**「直接戦うのがムリならば、間接的に戦えばいい」**という、**「弱いものいじめの法則」**に則ったランチェスター戦略のような戦略をとるべきです。

### 2. DCCM理論（差別性、優位性、説得性、市場性）

商品が売れるためには、**DCCM（差別性、優位性、説得性、市場性）**の4つの要素をチェックする必要があります。

| 要素 | 概要 | 課題・失敗の例 |
| :--- | :--- | :--- |
| **D（差別性）** | 競合商品との違い。 | 競合が強い市場に、差別優位性なく殴り込みをかける攻めの失敗。 |
| **C（優位性）** | 競合商品に対する優位性。 | 優位性には限界があり（例: スマートフォンの画面サイズと操作性）。 |
| **C（説得性）** | 差別性や優位性が消費者に伝わり、信じてもらえる度合い。 | 説得性がなければ優位性があっても商品は売れない。 |
| **M（市場性）** | 市場規模の大きさ。 | 差別優位性があっても**市場性が小さければ、本当のヒット商品にはならない**。市場拡大を狙いすぎると失敗する（例: キリン一番搾りのケース）。 |

### 3. トップブランドと劣位ブランドの戦略

*   **トップブランド（企業）の戦略:** 市場性が強く、優位性が高いことが特徴です。優位性を維持し、差別化を怠らないことが重要です。
*   **劣位ブランド（企業）の戦略:** 劣位ブランドは、差別優位性を強める戦略を取るべきです。劣位ブランドは、既存ジャンル内の強み（例: ハーゲンダッツはプレミアムアイス市場でトップ）を活かし、DCCM要素に基づき、差別化・優位性・説得性を高める戦略を取る。
*   **攻撃戦略と浸透戦略の組み合わせ:** 攻撃（正面、側面、ニッチ）と守り（スキミング、ペネトレーション）の戦略を理解し、状況に応じて組み合わせることが重要です。

## V. 商品の定義

### 1. プロダクトコーン理論

商品が生活者に提示する訴求内容は、**規格（スペック）**、**ベネフィット（機能的・心理的）**、**エッセンス（企業や個人の信念）**の3つの要素で定義されます。

*   **イノベーター理論との関係:** イノベーターは**規格**から入り、フォロワーは**エッセンス**から入ります。イノベーターは自己診断に基づいて商品を選択し、フォロワーは他己判断や人のススメ（イメージ）に基づいて選択します。
*   **パターン1（規格訴求せず失敗）**の失敗は、新製品導入時に生活者にまず「規格」を伝えていないことに起因します。

これらの理論は、企業がマーケティングの失敗から学び、成功確率を高めるためのツールとして提供されています。
