# Phase 2: 戦術選定

**親Skill**: marketing-strategy-planner
**Phase**: 2/3
**使用Skills**: b2b-marketing-60-tactics-playbook, marketing-framework-115-methods

## Purpose

Phase 1で生成されたWHO×WHATマトリクスを基に、最適なマーケティング戦術を選定する。b2b-marketing-60-tactics-playbook Skillの思考フレームワークを使用し、ファネル別・優先度別に施策を整理する。

## Thinking Framework

### b2b-marketing-60-tactics-playbook Skillの思考フレームワークを活用

このSubagentは、b2b-marketing-60-tactics-playbook Skillが提供する「BtoBマーケティング戦術選定思考」を使用して戦術を選定します：

1. **WHO×WHAT×HOWフレームワーク**
   - WHO: 価値を届けるべき人（Phase 1から）
   - WHAT: 戦略（Phase 1から）
   - HOW: 戦術（このPhaseで選定）

2. **ファネル別分類**
   - **潜在層**（認知・初期接触）: オウンドメディア、SNS広告等
   - **準顕在層**（興味・関心）: ナーチャリング、公式LINE、メルマガ、YouTube等
   - **顕在層**（比較・検討）: LP改善、5分以内の架電、サービス資料改善等

3. **施策優先順位の3つの観点**
   - **事業/経営視点**: 受注に近い順（顕在層 > 準顕在層 > 潜在層）
   - **顧客/業界別視点**: マーケチャネルを見極める（検索ボリューム、ビジネスモデル）
   - **ボトルネック別視点**: KPIとベンチマーク比較（CVR, アポ率、受注率）

4. **3つの○○を制する**
   - **頭の中を制する**: 認知率向上
   - **売り場を制する**: CVR改善
   - **使用体験を制する**: リピート率向上

### marketing-framework-115-methods Skillの思考フレームワークを活用

マーケティング手法の体系的な整理：

- 115種類のマーケティング手法から最適なものを選定
- WHO×WHATに基づいた手法選定
- 実行可能性・費用対効果を考慮

- 課題・ニーズ
- 便益・独自性

### Step 2: ファネル別戦術の選定

**b2b-marketing-60-tactics-playbook Skillの思考パターンを適用**:

1. **顕在層向け施策（最優先）**:
   - **LP改善**: CVR 150%アップを目指す（ファーストビュー最適化、CTAボタン改善）
   - **5分以内の架電**: アポ率400%改善
   - **サービス資料/営業資料改善**: 4つの不安を解消、サービスの見える化
   - **テレアポ改善**: アポ率20%以上を目指す、BANTCH評価

2. **準顕在層向け施策（中優先）**:
   - **ナーチャリング**: 商談化率150%を目指す（信頼獲得、継続接触、課題顕在化）
   - **公式LINE**: 開封率60-70%、クリック率10-20%（メルマガの10倍の効果）
   - **メルマガ**: 個別風メール、読者ニーズに合った配信
   - **YouTube**: 1企業1チャンネル、事例動画、トップセールスマン出演

3. **潜在層向け施策（低優先）**:
   - **オウンドメディア (SEO)**: キラーコンテンツ、1コンテンツマルチユース
   - **SNS広告/ディスプレイ広告**: 質の高いクリエイティブ

**優先度付けの基準**:
- **事業/経営視点**: 成果が早い、再現性が高い、予算コントロールしやすい → 顕在層優先
- **顧客/業界別視点**: 検索ボリュームの有無、ビジネスモデル（中小/エンタープライズ）
- **ボトルネック別視点**: KPIとベンチマーク比較
  - LP CVR基準値: 2%
  - アポ率基準値: 20%
  - 受注率基準値: 20%

### Step 3: 各戦術の詳細設定

**戦術リスト作成**:

| 戦術 | ファネル | 優先度 | 工数 | 期待効果 | 基準値 | 目標値 |
|------|--------|-------|-----|---------|-------|-------|
| LP改善 | 顕在層 | P0 | 2週間 | CVR 150%アップ | 2% | 3% |
| 5分以内の架電 | 顕在層 | P0 | 1週間 | アポ率400%改善 | 20% | 80% |
| サービス資料改善 | 顕在層 | P0 | 1週間 | 受注率向上 | - | - |
| 公式LINE | 準顕在層 | P1 | 3週間 | 開封率60-70% | - | 65% |
| YouTube | 準顕在層 | P1 | 4週間 | 受注率向上 | - | - |
| オウンドメディア | 潜在層 | P2 | 継続 | 認知率向上 | - | - |

**各戦術の実行可能性チェック**:
- リソース: 人員・予算が確保できるか
- タイムライン: 期限内に実行可能か
- 依存関係: 他の戦術との依存関係はあるか

## Troubleshooting

### Phase 1の01_who_what.mdが見つからない

**原因**:
- Phase 1が実行されていない
- ファイルパスが間違っている

**対処**:
```bash
# Phase 1の成果物確認
ls -la /Users/ksato/workspace/_codex/projects/<project>/marketing/01_who_what.md
```

存在しない場合は、Phase 1を先に実行

### 戦術選定が不適切

**原因**:
- Phase 1のWHO×WHATマトリクスが不明確
- ファネル別分類が不十分
- 「受注に近い順」の優先順位付けができていない

**対処**:
- Phase 1に戻ってWHO×WHATマトリクスを修正
- ファネル別（潜在層/準顕在層/顕在層）に明確に分類
- 顕在層向け施策を最優先に設定

### 戦術数が多すぎる（10個以上）

**原因**:
- 戦術の粒度が細かすぎる
- 優先順位が不明確

**対処**:
- 戦術を統合（関連施策をまとめる）
- 優先度の低い戦術を削除（P2は削除候補）
- 最優先戦術（P0）を5個以下に絞る

### 実行可能性が低い

**原因**:
- リソース・予算が不足
- タイムラインが非現実的

**対処**:
- AskUserQuestionでリソース・予算を確認
- 戦術の優先順位を見直し
- 段階的な実行計画を立てる（Phase 3で調整）
