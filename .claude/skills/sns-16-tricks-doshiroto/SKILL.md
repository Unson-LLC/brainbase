### A. コラボ禁止！アカウント立ち上げ初期に絶対やってはいけないこと

SNSで成功を掴めるかは、最初の数投稿で8割が決まる。

**立ち上げ初期は"アルゴリズムの教育期間"**:
- AIに自分のアカウントの正体を学ばせるフェーズ
- 「誰に」「どんな内容を」届けたいのかを定める

**絶対にやってはいけないこと**:
- コラボ
- 相互フォロー
- 社内共有
- 友人への宣伝

**理由**: 外部からの不純な信号がAIを混乱させる

**例**: 美容アカウントを始めた人が友人に宣伝
→ 美容に関心のない友人が反応
→ AIが誤認「このアカウントは学生男子や主婦層向け」
→ 視聴維持率低下 → 伸びないアカウントに格下げ

**やるべきこと**: 統一と継続
- 発信ジャンルとフォーマットを統一
- 毎回同じ世界観で投稿
- 最初の100フォロワーは「友達」ではなく「未来の顧客」

### B. ど素人ホテル式、SNSでの最適ペルソナ設定

**逆転の発想**: 「あえて自社商品に全く興味がない人」をペルソナに設定

- レコメンド型SNSは"探す場所"ではなく"偶然出会う場所"

**見込み客を狙うほど動画は伸びない**:
- 沖縄ホテルが「沖縄旅行を計画している人」をターゲット → 失敗

**あえて"絶対に来ない人"をペルソナに**:
- 沖縄ホテル → 「青森の山奥に住むおじさん」
- 焼肉屋 → ビーガン
- 化粧品 → スキンケアに無頓着な男子高校生

**理由**: 最も興味がなさそうな人が離脱しない動画 = 誰でも最後まで観る → AIが良質と判断 → バズ

**"情報動画"を"エンタメ動画"に変換する**:
- 情報：「レンタカー無料企画1年延長！」→ 誰も見ない

## 台本制作の黄金法則

### 自己紹介から始める台本

**例**: 「僕はいきなり沖縄の大赤字ホテルを再建するど素人」

- 「いきなり」という言葉で"ツッコミたくなる"感情トリガーを引く
- ツッコミたくなる = 脳が能動的になっている
- 自己紹介は最も自然に"離脱を防ぐ冒頭"

### フォロワーが増えたらキャッチフレーズ導入に切り替え

- 毎回同じ自己紹介 = 離脱の原因になる
- キャッチフレーズ導入 = 「誰が見ても思わず止まる一言」で始める
- 正しく伝える必要はない。誤解されるくらいでちょうどいい

### 「誤解→共感→回収」の台本構造

1. **誤解を生む**（見込み外層を止める）
   - 例：「ホテル辞めます」「倒産しました」「ごめんなさい」
2. **共感でつなぐ**（視聴者の気持ちを代弁）
   - 例：「今まで応援していただきましたが、今回は無理そうです」
3. **回収で納得させる**（真実を明かして感情を整理）
   - 例：「辞めるのは今日だけです」

## フォロワー参加型の仕組み

視聴者を"観客"から"当事者"に変える

**具体的な方法**:
- ホテル名オーディション
- ロゴ募集
- お仕事依頼報告
- コメント、アンケート、ライブ配信での投票

**効果**:
- 「自分もこの挑戦の一部になっている」という感覚
- コメントやシェアが伸びる
- 「少しでも経営に関われた」と思う関係人口が増加
- 購買数が圧倒的に変わる

## まとめ

1. **プロセスエコノミー**: 未完成であることが最強のブランド
2. **正しい弱者設定**: 目標に対して正しいベクトルで弱者であること
3. **ワンチャン感**: 確実にできそうと絶対にできなそうの間
4. **台本**: 誤解→共感→回収の3ステップ
5. **参加型**: 視聴者を当事者に変える
6. **透明化**: 正直さ×透明化×仕組み化が最強のマーケティング
