## コア戦略

### プロセスエコノミーの設計

**定義**: 結果ではなく過程をコンテンツ化

**ワンチャン感の方程式**:
```
目標 × それに対する弱者 × ワンチャンいけるかもしれない特殊技術 = バズ
```

**正しい弱者設定のルール**:
- 目標に対して正しいベクトルで弱者であること
- 「確実にできそう」と「絶対にできなそう」の間を狙う
- 身長とホテル経営のような無関係な弱者設定はNG

**例**:
- 大赤字ホテルを黒字化 × ど素人経営者 × SNSは上手い → ワンチャン感あり
- NBA選手になる × 140cm × ドリブル技術は天才 → ワンチャン感あり

## ABCDEFGH戦略

### A. 立ち上げ初期の禁止事項

**コラボ・相互フォロー・社内共有禁止**

**理由**: 外部からの不純な信号がAIを混乱させる
→ 美容アカウントを友人に宣伝 → 美容興味ない層が反応 → AIが誤認 → 伸びない

**やるべきこと**: 統一と継続
- 発信ジャンルとフォーマット統一
- 最初の100フォロワーは「未来の顧客」

### B. 最適ペルソナ設定

**逆転の発想**: あえて自社商品に全く興味がない人をペルソナに

**例**:
- 沖縄ホテル → 青森の山奥おじさん
- 焼肉屋 → ビーガン
- 化粧品 → スキンケア無頓着な男子高校生

**理由**: 最も興味ない人が離脱しない = 誰でも最後まで観る → AIが良質と判断

### C. 日常を事件に変換

**1,353万再生のキャッチフレーズ**:
- 本来: 「消防点検が来る日」
- 変換: 「私のホテルが法律違反をしています」

**3構造**:
1. 感情を動かす強ワード
2. 実際の事実に紐づける（釣りではない）
3. 本文で伏線回収

### D. 演者ランキングTOP7

1. **社長**: 信頼・採用・営業すべてに効果
2. **街の人々**: リアルな人間模様（要同意書）
3. **引退した創業社長**: 最強のストーリーテラー
4. **社内で強烈な特徴を持つ人**: 身長2m超え、80歳現役等
5. **新入社員**: 成長ストーリーで採用動画と相性抜群
6. **社長の家族**: 子どもとの共演でギャップ
7. **社長秘書**: 社長の時間を取らず撮影可能

## 台本制作の黄金法則

### 自己紹介から始める

**例**: 「僕はいきなり沖縄の大赤字ホテルを再建するど素人」
- 「いきなり」でツッコミたくなる感情トリガー
- ツッコミたくなる = 脳が能動的

### フォロワー増加後はキャッチフレーズ導入

- 毎回同じ自己紹介 = 離脱の原因
- 誤解されるくらいでちょうどいい

### 「誤解→共感→回収」の構造

1. **誤解を生む**（見込み外層を止める）
   - 例：「ホテル辞めます」「倒産しました」

2. **共感でつなぐ**（視聴者の気持ちを代弁）
   - 例：「今まで応援していただきましたが、今回は無理そうです」

3. **回収で納得させる**（真実を明かす）
   - 例：「辞めるのは今日だけです」

### ビジュアルフック

**最初の2秒**で感情トリガーを引く
- サッカー選手: 崖でリフティング → ボールが落ちそうで心配
- 白い服で担々麺をすする → 「汚れちゃう！」という感情

## プロセスエコノミー収益化

**なぜ物が売れるのか**:
- チャンネル全体が一つの物語
- 視聴者は「人生ドキュメンタリー」を見ている
- 接触時間が長く、感情移入が深い
- 商品が"ストーリーの一部" → 買うことで物語に参加

**道の駅のジャガイモの例**:
- ただのジャガイモ → 誰も買わない
- 「おじいさんが30年かけて作った」 → 手が伸びる

## フォロワー参加型の仕組み

視聴者を"観客"から"当事者"に変える

**具体的な方法**:
- ホテル名オーディション
- ロゴ募集
- お仕事依頼報告
- コメント、アンケート、ライブ配信での投票

**効果**:
- 「自分もこの挑戦の一部」という感覚
- コメント・シェアが伸びる
- 購買数が圧倒的に変わる

## まとめ：6つの原則

1. **プロセスエコノミー**: 未完成が最強のブランド
2. **正しい弱者設定**: 目標に対して正しいベクトルで弱者
3. **ワンチャン感**: 確実にできそうと絶対にできなそうの間
4. **台本**: 誤解→共感→回収の3ステップ
5. **参加型**: 視聴者を当事者に変える
6. **透明化**: 正直さ×透明化×仕組み化
