# Phase 3: Finalize（最終調整）

## Mission

**note記事の「公開準備」を自動化**

Phase 2.5のレビューを通過したドラフトに対し、見出し画像プロンプト生成、メタ情報設定、公開チェックリストを作成し、公開可能な状態に仕上げます。

## Context

**Phase 2.5のレビュー結果**: `_codex/sns/drafts/{topic}_reviewed.md`
**Phase 2のドラフト**: `_codex/sns/drafts/{topic}_draft.md`

**参照ファイル**:
- `_codex/sns/note_strategy.md` - note戦略
- `_codex/sns/x_account_profile.md` - アカウント情報

## Task

以下のタスクを実行してください：

### 1. 見出し画像プロンプト生成

**nano-banana-pro-tipsの思考フレームワークを適用**

note記事の見出し画像（ヘッダー画像）を生成するためのプロンプトを作成します。

**画像の目的**:
- 記事の内容を一目で伝える
- 読者のクリック意欲を高める
- ブランドイメージを統一する

**画像タイプの選定**:

| タイプ | 適用シーン | 例 |
|--------|-----------|-----|
| **インフォグラフィック** | 数字・データを視覚化 | 「20個→100個」の進捗バー |
| **進捗/Before-After** | プロセスエコノミー記事 | 「Day 1」のタイムライン |
| **事件風サムネ** | 衝撃的な内容 | 「危険すぎる」のテキストオーバーレイ |
| **概念図/フロー** | ノウハウ記事 | 自動化フローの図解 |

**プロンプト作成のポイント**:
1. **文字描写**: 日本語テキストを正確に画像内に描写
2. **構図指定**: 横長レイアウト（16:9アスペクト比）
3. **禁止事項**: オチやネタバレを含めない
4. **スタイル**: ビジネス × プロフェッショナル

**プロンプトテンプレート**:
```
日本語の[タイプ名]ビジュアルを作成してください：

テーマ: [記事のメインテーマ]

デザイン要件:
- [視覚化する要素1]
- [視覚化する要素2]
- [視覚化する要素3]
- 横長レイアウト（16:9アスペクト比）

含めない要素:
- [含めてはいけない要素]

スタイル: [ビジネス/プロフェッショナル/カジュアル等]

重要: すべて日本語で表示してください。
```

---

### 2. メタ情報の設定

#### タイトル確定

- Phase 1で決定したタイトルを確認
- 本文との整合性を確認
- 必要に応じて微調整

#### タグ選定

**タグの原則**:
- 5-10個程度
- 関連性の高いタグを優先
- 人気タグ + ニッチタグの組み合わせ

**タグカテゴリ**:

| カテゴリ | 例 |
|---------|-----|
| **チャレンジ系** | #100日チャレンジ, #毎日更新, #挑戦 |
| **業務系** | #PM業務, #プロジェクトマネジメント, #業務効率化 |
| **技術系** | #AI活用, #自動化, #Claude, #ChatGPT |
| **働き方系** | #リモートワーク, #副業, #起業 |
| **マインドセット** | #習慣化, #継続, #失敗から学ぶ |

#### 公開日時推奨

**noteの最適投稿時間**:
- **平日**: 7:00-8:00（通勤時間）、12:00-13:00（昼休み）、21:00-22:00（就寝前）
- **休日**: 10:00-11:00（遅めの朝）、21:00-22:00（夜）

**推奨日時の決定基準**:
- ターゲットペルソナの生活リズム
- 曜日別のエンゲージメント傾向
- 競合記事の投稿タイミング

### 5. X告知文の準備

note記事公開後にXで告知するための文案を準備：

**X告知文のテンプレート**:
```
[フック: 記事のメインメッセージを1行で]

[本文: 記事を読むメリットを2-3行で]

記事はこちら👇
[note URL]

#[タグ1] #[タグ2]
```

**ポイント**:
- 140文字以内（Xの制限）
- 記事を全部読まなくても価値がわかる
- CTAは「詳細はこちら」ではなく具体的に

## Important Notes

1. **見出し画像の重要性**: 見出し画像がある記事はビュー数・スキ数が数十%向上
2. **タグの最適化**: 人気タグ + ニッチタグの組み合わせでリーチを最大化
3. **X連携**: note公開後すぐにXで告知（1時間以内がベスト）
4. **継続性の確保**: 次回記事の構想メモを残しておく
