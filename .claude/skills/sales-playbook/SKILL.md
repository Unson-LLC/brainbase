## § 1. セールスの基本原理

### 1.1 購買の本質：ベネフィット

**原則**: 顧客は商品ではなく、**ベネフィット（嬉しい未来・結果・幸福）**にお金を払う

**ベネフィットとは**:
- 商品から得られる嬉しい未来（結果、幸福）
- 特徴やメリットは、ベネフィットが叶う**理由**

**思考パターン**:
```
商品特徴を列挙 → 「ということはつまり？」を繰り返す → ベネフィットを導出
```

**Example**:
```
❌ 特徴: 「高速CPUを搭載」
✅ ベネフィット: 「動画編集が3倍速くなり、家族との時間が増える」

❌ メリット: 「営業ナレッジを一元化」
✅ ベネフィット: 「新人が3日で商談できる状態になり、育成工数80%削減」
```

---

### 1.2 成功の80%は書く前に決まる

**原則**: 広告の成否は、文章を書き始める前の段階で80%決まる

**Why**:
- 売れない最大の理由は「読まれていない」
- 顧客に興味がなければ、どんな優れた文章も無意味

**思考パターン**:
```
考える（80%） → 表現する（20%）
↓
「欲しい人へ、欲しいものを売る」ルールを徹底
```

**重要な発見**:
- コピーライティングは「文章表現をこねくりまわすこと」ではない
- 思考を深めることが最も重要

---

### 1.3 購買意欲の5段階

**原則**: 顧客の心理は「認知→興味→欲求→確信→決断」の5段階で変化する

**5段階モデル**:
```
1. 認知（Awareness）: 商品を知る
   → アプローチ: 心を開く

2. 興味（Interest）: 少し知っているが欲しくない
   → ヒアリング: 問題・願望を引き出す

3. 欲求（Desire）: ベネフィットに興味あり
   → プレゼンテーション: 地獄の情景を見せる

4. 確信（Conviction）: 価値を確信
   → クロージング: 訴求する

5. 決断（Decision）: 購入を決める
   → 切り返し: 決断させる
```

**思考パターン**:
```
顧客の現在段階を特定 → 次の段階に進めるアクションを選択
```

---

### 1.4 ターゲットの3タイプ分類

**原則**: 購入意欲の強さに応じて3タイプに分類し、訴求を変える

**3タイプ**:

| タイプ | 状態 | 判断基準 | 訴求方針 |
|--------|------|----------|----------|
| **タイプ①（関心客）** | めっちゃ興味を持っている | ブランドの強さ、信頼関係、超強力オファー | 商品名+魅力的なオファーをシンプルに見せる |
| **タイプ②（検討客）** | 少し知っているが、まだ欲しくない | 差別化要因、他より良いオファー | ズバ抜けた違いを語り、確信させる |
| **タイプ③（潜在客）** | ベネフィットに興味あるが商品を知らない | 差別化が難しい、オファーが弱い、わかりにくい | 商品名を語らず、最高の解決策を提案 |

**思考パターン**:
```
ターゲット分類 → タイプ別訴求を設計 → 適切なトーンで伝える
```

---

### 1.5 商品認知の4ステージ

**原則**: ターゲットの「商品に対する認知度と欲求の高さ」で4ステージに分類

**4ステージ**:

| ステージ | 状態 | 欲求レベル | アプローチ |
|----------|------|-----------|-----------|
| **① その商品が欲しい** | めっちゃ興味を持っている | 最も高い | リスティング広告、LP |
| **② 少し知っているが欲しくない** | 興味はあるが購入を迷っている | 高い | LP（徹底的な説明） |
| **③ ベネフィットに興味あるが商品を知らない** | 漠然とした悩みを持つ | 中程度 | Facebook広告、コンテンツマーケ |
| **④ まったく無関心** | 興味なし | 低い | アプローチ困難 |

**思考パターン**:
```
媒体選定 → ターゲットステージ特定 → 適切なコピー設計
```

## § 2. 非対面セールスコピー設計

### 2.1 売れるアイデアの捉え方

**原則**: 売れるアイデアは「生み出すもの」ではなく「拾うもの」

**3ステップ**:
1. お客さんの脳内にある「売れる世界」を探す
2. その世界で求められる「隠れた提案」を考える
3. その提案を魅力的に語る（コピーを書く）

**思考パターン**:
```
顧客の脳内を探索 → 隠れた提案を発見 → 魅力的に表現
```

---

### 2.2 ターゲット別「ドンピシャ訴求」

**原則**: ターゲットの感情・知識レベルに合わせて訴求を変える

**タイプ別訴求**:

**タイプ①（関心客）への訴求**:
- 商品名と魅力的なオファーをシンプルに見せる
- 例: 「iPhone 15 Pro、今なら3万円キャッシュバック」

**タイプ②（検討客）への訴求**:
- ズバ抜けた違いを語り、確信させる
- 例: 「他社の2倍の速度、業界唯一の全額返金保証」

**タイプ③（潜在客）への訴求**:
- 商品名や価値を語らず、ベネフィットが手に入る最高の解決策を提案
- 例: 「営業が苦手なあなたでも、3日で商談できるようになる方法」

**売りにくい商品（タイプ③）を売る2つの方法**:
1. **商品特徴から新たなターゲットを探す**: 特徴・メリットから「欲しい人」を探し出す
2. **ターゲットの悩みから新たな訴求を探す**: 強い悩み・欲求を洗い出し、解決に繋がるベネフィットを考える

---

### 2.3 コピーの3構成要素

**原則**: 広告文は「キャッチ」「リード」「ボディ」の3つの世界で構築

#### 2.3.1 キャッチコピー（Catch Copy）

**目的**: 読み手の注意を一瞬で掴み、続きを読ませる

**初心者でもできる4ステップ法**:
1. ムダな言葉を省く（ターゲティングコピーを削る）
2. 長文を分割しリズムよく（短文に分割）
3. 不要な言葉を徹底削除（ギリギリまで削る）
4. 問いかける（疑問形で興味を引く）

**売れるキャッチコピー「13の表現法」**:
1. ベネフィットを語る（具体的に、現実的に）
2. 読み手を絞り込む
3. 数字を上手に使う（具体的数字、単位の制御）
4. 結果・実績を見せる
5. ビフォーアフター
6. 社会的証明
7. 権威を見せる
8. カンタン、すぐに、誰でもできる
9. 五感に訴える
10. 続きを読ませる（ザイガニック効果）
11. 驚きを読ませる
12. 魅力的なオファーを語る
13. 禁止する（カリギュラ効果）

**キャッチコピー10のテクニック**（pochiruseru版）:

| テクニック | 概要 | 効果 |
|-----------|------|------|
| ① ベネフィットを語る | ハッピーな未来を伝える | 広告表示に必須 |
| ② 読み手を絞り込む | ターゲットを具体的に | 心に響く |
| ③ 驚きを読ませる | 意外な発見を提供 | ザイガニック効果 |
| ④ 魅力的なオファー | 無料、保証、特典 | 今すぐ契約できる条件 |
| ⑤ 禁止する | 敢えて行動を禁止 | カリギュラ効果 |
| ⑥ 数字を上手に使う | 具体的な数字・単位 | 信頼性向上 |
| ⑦ 証拠を見せる | 社会的証明、推薦 | 権威性・信頼性 |
| ⑧ 結果を見せる | ビフォーアフター | 劇的な変化を視覚化 |
| ⑨ 五感に訴える | 目・耳・鼻・舌・肌 | 快楽をイメージさせる |
| ⑩ ストーリーを語る | V字型構造 | 感情移入、記憶に残る |

**効果的なキャッチコピーのテンプレート**:

| テンプレート | 構成 | 例 |
|------------|------|-----|
| ① 意外性の高い提案 | 意外性+ベネフィット+方法 | 10年悩んだ腰痛を3日で劇的改善した方法 |
| ② ストーリー性 | 不利な状況+成功 | 学年ビリが国立大学に現役合格できたのか？ |
| ③ 強力なベネフィット | 強烈なベネフィット+約束 | 3ヶ月でマイナス10キロ痩せなければ全額返金 |
| ④ 新規見込み客へ | 状況+悩み+あなたへ | イヤイヤ期の子供を前向きのママへ |
| ⑤ 不安をあおる | 間違いをしていませんか？ | ダイエットでリバウンドする間違いをしていませんか？ |

---

#### 2.3.2 リードコピー（Lead Copy）

**目的**: 注意を掴んだ読み手を、続きが気になる状態に保ち、最大値まで引き上げる

**3つの要素**:
1. キャッチコピーの強化
2. 続きが知りたくなる（ザイガニック効果）
3. 魅力的なオファー

**思考パターン**:
```
キャッチで注意 → リードで興味最大化 → ボディで購入意思育成
```

---

#### 2.3.3 ボディコピー（Body Copy）

**目的**: 読み手に購入の意思を育て、広告の3原則（①読まない、②信じない、③行動しない）を突破

**売れるボディコピーの5つの法則**:
1. 「1対1」を意識する（「1対多」ではない）
2. 「あなた」を意識する（「私」ではない）
3. 伝えたいことは「1つ」に絞る
4. 投げやりな言葉は使わない
5. 「わかる」ではなく「気づかせる」

**ボディコピー作成の5ステップ**:
1. 読み手に共感
2. 問題提起と解決条件の提案
3. 具体的な解決策の提案
4. ベネフィットの訴求
5. クロージング（今すぐ申し込む理由、リスクリバーサル、価格の正当性）

**売れるボディコピーの6ステップ**（pochiruseru版）:
1. 煽り効かせる（キャッチコピー）
2. 近づいてもらう（リードコピー）
3. 信用できる証拠を見せる
4. ベネフィットを伝える
5. ベネフィットが叶う理由として商品を伝える
6. 今すぐ動いてもらう理由や条件を伝える

**販売力を強化する「21の表現技術」**:

**説得力**:
- 客観的な事実を語る
- 具体的に語る
- 証拠を見せる
- リンゴとミカンで比較（価格・価値の正当性）
- 3つの理由を述べる
- 三段論法

**わかりやすさ**:
- 具体化+アポ化
- 読み手が使う言葉で語る
- 漢字を減らす（ひらがな比率70%推奨）
- 一文を短くする
- 情報を視覚化（図・イラスト）

**続きを読ませる**:
- リズム感を高める
- 小見出しを効果的に使う
- 問いかける

---

### 2.4 ストーリーテリング

**原則**: 特に購入意欲が低いターゲット（タイプ③）に有効

**売れるストーリーの3つの条件**:
1. ターゲットが共感できる主人公を設定
2. V字型（日常→どん底→成功）のストーリー展開
3. ストーリーとセールスが美しくつながる構成

**重要**: ウソのストーリーはダメ。事実を魅力的に語る技術を使う

---

### 2.5 売れる「オファー」の作り方

**オファーの定義**: お客様と約束する魅力的な取引条件（特典、保証、価格など）

**売れるオファーの6つのタイプ**:
1. 価格オファー
2. リスクリバーサル（返金保証等）
3. 特典オファー
4. 無料オファー
5. 時限オファー
6. 利便性オファー

**成功するオファーの3条件**:
1. 大胆で、ありえないオファー
2. その業界で初めてのオファー
3. 客の欲求を満たし、悩みや不安を消すオファー

---

### 2.6 検証とデザイン

#### 2.6.1 科学的に売れるコピーを導く「広告テスト法」

**原則**: セールスコピーは科学。正しい広告テストでレスポンス最大化

**スプリットテスト（ABテスト）の原則**:
- テストで変える要素は**1つだけ**
- 「正解」を見つけたら、効果がなくなるまで徹底的に使いまわす

**ポイント**:
- 一度に1ヶ月だけ、特定の条件を変えて比較
- 初心者はまずキャッチコピーを変更してテスト

---

#### 2.6.2 読みやすくなる「レイアウトと装飾」

**原則**: KISS（Keep It Simple Stupid）を頭に置き、シンプルなデザインに

**技術**:
- アイフロー（Z型・N型）を意識
- 四角枠で囲む
- 文章にメリハリをもたせる
- 目を留める画像

## § 3. 対面営業5ステップ

### 3.1 営業の本質

**原則**: 営業は、お客様の「購買意欲」を高める作業

**トップセールスの「暗黙知」の言語化**:
- 経験や勘に頼る「暗黙知」を整理
- 「型」として形式知化
- 「すごい営業」を誰もが実現できる状態に

**言語化による効果**:

**個人レベル**:
- 売上と精神的な安定
- 意欲が湧き、「できる営業」だと確信

**組織レベル**:
- 「売れる営業」の技術に再現性
- 新人もすぐに高い結果
- ベテランはさらに売上を伸ばす
- 離職を防ぐ

---

### 3.2 第1段階：アプローチ（相手の心を開く）

**目的**: 相手の心の扉を物理的に開く

**心構え**: 人間は初対面の相手を最初の6秒間で「敵か、味方か」判断

**主要技術**:

**褒める（2段階）**:
- 褒めて「心地よさ」を感じてもらう
- 幸福ホルモン（エンドルフィン）を分泌させる

**同意する（100％同意）**:
- 話の「内容」ではなく「態度」に対して100％同意
- 好意を持ってもらう

**オウム返し**:
- 相手の言葉を繰り返す
- 話を「聴いていること」を伝える
- 信頼と幸福ホルモンを増やす

**思考パターン**:
```
最初の6秒 → 「敵か、味方か」判断される → 褒める+同意+オウム返しで心を開く
```

---

### 3.3 第2段階：ヒアリング（問題や願望を引き出す）

**目的**: 顧客自身に「問題」や「願望」を語らせ、自覚させる

**心構え**: 「営業は効果本位で考える」。お客様自身に「ついしゃべっちゃった」と思わせるトーク

**主要技術**:

**限定質問**:
- イエス・ノーで答えられる質問から入る
- お客様に楽をさせる
- 状況把握、ヒアリングの入り口を作る

**第三者話法**:
- 他の人の事例を用いる
- 客観的な事実として共感を引き出す
- 自身の問題ではないと思わせる

**実績を付け加える**:
- 具体的な数字やデータで信頼性を高める
- 話に説得力を持たせる

**YES & SO（「それでしたら」）の使用**:
- 相手の言葉を否定せず受け止める（YES）
- そこから話題を購買意欲へ向ける（SO）

**思考パターン**:
```
限定質問で状況把握 → 第三者話法で共感 → YES & SOで購買意欲へ誘導
```

---

### 3.4 第3段階：プレゼンテーション（感じてもらう）

**目的**: 顧客に「地獄の情景」を思い浮かべてもらい（恐怖回避の心理）、購買意欲を確信へ導く

**心構え**: 人は「理論」ではなく、「感情」で動く

**主要技術**:

**利点話法**:
- 商品の特徴（セールスポイント）ではなく
- **お客様にとっての具体的なベネフィット（利点）**を伝える

**比較話法**:
- 人間がすべてを比較で判断する心理を利用
- 「他社」「相場」「初期段階」の3つの要素と比較
- 価値を明確にする

**IF活用**:
- 顧客に「それだったら、欲しいよね」という感情を揺さぶる
- YESを取るための誘導尋問

**思考パターン**:
```
利点話法でベネフィット → 比較話法で価値明確化 → IF活用で感情を揺さぶる
```

---

### 3.5 第4段階：クロージング（訴求する）

**目的**: お客様に対して「契約してください」と**明確に訴求**

**NG行為**: 契約を決めずに「お伺い」や「委ねる」形で保留にすること

**主要技術**:

**料金説明の鉄則（都合のいい比較）**:
- 競合他社や自社製品を比較
- 価格が高くても「価値が高い」と感じさせる

**二者択一**:
- 複数の選択肢を提示
- お客様に「判断を減らす」
- 迷いをなくす

**バンドワゴン効果**:
- 「みんながやっている」という人気を利用
- 無意識下の購買意欲に働きかける

**ウィンザー効果**:
- 第三者や専門家の意見を引用
- 信頼性を高める

**一貫性**:
- お客様の発言から「一貫性」のロジックを導き出す
- 決断に繋げる

**思考パターン**:
```
料金説明で価値納得 → 二者択一で判断簡易化 → 心理効果で決断促進
```

---

### 3.6 第5段階：決断への誘導/切り返し（決断させる）

**目的**: お客様の「考えます」という保留（多くはウソ）を突破し、決断を誘導

**心構え**: 顧客の「考えます」に対し、共感と感謝を伝えてから切り返す

**主要技術**:

**質の高い切り返しを複数回**:
- 「切り返し」は1回では無効
- **最低3回**、できれば5回連続で続ける
- お客様の「一貫性」を崩す

**AREA話法の応用**:
- 主張（A）、理由（R）、具体例（E）、再主張（A）
- 論理構成を用い、切り返しトークを効率的かつ効果的に

**すり替えの技術**:
- 表向きの迷い（相談したい、他社と比較したい）を
- 「私とあなたの問題」（解決を先送りすることのデメリット）に焦点をすり替え
- 決断を促す

**「価値の上乗せ」の技術**:
- 「高い」という価格への懸念に対し
- 商品の価値を再主張
- 価格が高いことの理由を納得させる

**思考パターン**:
```
「考えます」を受け止める → 切り返し3〜5回 → AREA話法で論理構成 → 決断へ誘導
```

## § 4. オンライン提案作成

### 4.1 「読まれる提案」作成の3ステップ

**原則**: 欲しい人へ、欲しいものを売る

**3ステップ**:

**ステップ1: 商品特徴やメリットの徹底的な洗い出し**
- 目の前の商品の特徴やメリットを徹底的に書き出す

**ステップ2: 既存ターゲット以外で「欲しい人」を探す**
- 商品が持つ特徴やメリットに
- **既存のターゲット層「以外」で**強く反応する「欲しい人」を探す
- 視点を変えることで、欲求の高いターゲット層（「泳ぎ」）を発見

**ステップ3: 強烈に反応するベネフィットを考える**
- 「欲しい人」が、その商品を入手することで得られる
- **「ハッピーな未来」**、すなわちベネフィットを明確に
- 抽象的ではなく、**具体的**にイメージしやすいように書く

**思考パターン**:
```
商品特徴洗い出し → 既存以外のターゲット探索 → 強烈なベネフィット設計
```

---

### 4.2 LPの構成要素

**原則**: LP（ランディングページ）は「最強のセールスマン」として機能

**3つの基本要素**:
1. **大前提**: 1つの商品や1つの提案だけを徹底的に解説
2. **キャッチコピー**: 最初に必ず必要
3. **ボディコピー**: ステップに沿って商品を詳細に説明
4. **申し込みフォーム**: 最後にお客様がすぐに申し込めるように配置

**LPの目的**:
- ステージ②のターゲット（少し知っているが欲しくない）に
- 商品の魅力を徹底的に説明し、購入を促す

---

### 4.3 ネット媒体ごとの戦略

**原則**: 媒体ごとにコピーのツボが異なる

**媒体別アプローチ**:

| 媒体 | 主なターゲットステージ | アプローチのポイント |
|------|----------------------|---------------------|
| **リスティング広告** | ①、②、③ | キーワード設定で複数ステージにアプローチ |
| **LP（ランディングページ）** | ② | 最強のセールスマンとして徹底説明 |
| **Facebook広告** | ③ | セグメント（属性）で絞り込み、悩みに直結 |
| **リマーケティング** | ③ | 再訪問促進、LPへ誘導 |
| **類似オーディエンス** | ③ | 近いタイプのユーザーを自動探索 |
| **メルマガ** | 既存顧客・見込み客 | 件名に「ある要素」で開封率向上 |

**思考パターン**:
```
媒体選定 → ターゲットステージ確認 → 適切なコピー設計
```
