## Triggers

以下の状況で使用：
- B2Bのリード〜商談〜受注〜CSまでのハンドオフ基準を整備するとき
- レベニュー組織のボトルネック診断やKPI設計を行うとき

この本に書かれている本質的な内容は、「**マーケティング・インサイドセールス・営業・カスタマーサクセスの共業プロセス**」という「ザ・モデル」の概念に基づき、組織の構造化、プロセスの管理、そして成果を最大化するための戦略と人材育成について、体系的に解説したものです。

以下に、その本質的な要素を構造化して詳細にまとめます。

---

## Ⅰ. 「ザ・モデル」の核となる概念

### 1. ビジネスにおける「再現性」の追求
ビジネスにおいて最も重要なことの一つは「再現性」であり、本書はこの再現性を実現するための体系的なプロセス管理手法として「ザ・モデル」を紹介しています。この手法は、成功に至る過程を体系化し、実践的な知識をまとめた「プレイブック」として活用されることを目的としています。

### 2. 営業プロセスの分業から共業へ
従来の営業プロセスは、一人の営業担当者がターゲットリスト作成から受注まで、顧客接点のすべてをカバーするものでした。
しかし、「ザ・モデル」では、効率的かつ持続的な成長のために、営業プロセスを専門機能ごとに**分業**し、部門間の連携（**共業**）を重視します。

**プロセスを構成する主要部門とその役割**:

| 部門 | 主な役割 |
| :--- | :--- |
| **マーケティング** | リード獲得と認知拡大。ウェブサイト、キャンペーン、セミナーなどを通じた活動量の最大化。 |
| **インサイドセールス** | リードの育成（ナーチャリング）と、商談への移行（クオリファイ）を行う**商談供給の調整弁**。 |
| **営業 (AE/フィールドセールス)** | 有望リードに対する提案、交渉、契約プロセスを管理し、受注を獲得する。 |
| **カスタマーサクセス (CS)** | 顧客の活用促進、リテンション、アップセル/クロスセルを支援する。 |

## Ⅱ. リードと商談のプロセス管理

### 1. リードの段階的評価（顧客ステージ）
顧客が購入に至るまでの検討プロセスが変化した現代では、営業が接点を持つ前の「調査・評価」プロセスが重要度を増しており、これを細かく管理します。

**顧客ステージ（マーケティング/インサイドセールス担当範囲）**:

1.  **認知拡大:** ウェブサイト訪問数、イベント参加者数など。
2.  **リード獲得:** メールアドレスなどのコンタクト情報の取得。
3.  **リード育成:** メールリンクのクリックや製品サイトへのアクセス、セミナー参加など、興味関心のある行動を促進。
4.  **有望リード:** 商品カタログ請求、問い合わせ、またはリードスコアが100点以上など、購買意欲が高まった状態。

### 2. リサイクル（Recycle）の重要性
新規リードの約65%は「将来購入の可能性はあるが今すぐではない」という見込み客であり、ナーチャリング（継続的フォロー）が必要です。

*   **失注・未商談リードの価値:** 受注・未商談リードは、**コストをかけずに獲得できる商談機会**の「宝の山」となり得るため、マーケティングオートメーション（MA）を活用した再商談化（リサイクル）の流れを作ることが極めて重要です。
*   **プロセスの再構築:** 効率を追求する中で限界が訪れる問題（例えば、売上を2倍にするためにリード件数を4倍にする必要があるなど） は、このリサイクルルート（迂回回路）を追加することで解決されます。

### 3. 営業のフェーズ管理
営業活動は、以下の5つのフェーズ（商談ステージ）で進行します。

| フェーズ | 確度 | 定義 (エッセンス) | 移行判定基準 (例) |
| :--- | :--- | :--- | :--- |
| **フェーズ 1** | 0% | リード以上、商談未満の状態。 | 自社製品やサービスの解決可能性、商談未満の状態でいること。 |
| **フェーズ 2** | 25% | 顧客のビジネス課題を認識し、自社の解決手段を理解する。 | BANT条件（予算、決裁権、必要性、導入時期）を確認できたこと。 |
| **フェーズ 3** | 30% | 評価と選定。役職など内部の影響力を持つ人と合意し、自社製品を選定してもらう。 | 競合他社との差別化や優位性に基づいて、顧客に選択条件を自ら作り出すことが必要。 |
| **フェーズ 4** | 50% | 最終交渉と意思決定。顧客と合意した提案書や契約に必要なタスクを実行。 | 正式に稟議プロセスを開始し、契約を締結するための道筋が整うこと。 |
| **フェーズ 5** | 75% | 稟議決裁プロセス。最終プロセスでのリスク管理、稟議通過。 | Mutual Close Plan (顧客と相互に合意した計画) の完了。 |

## Ⅲ. 三つの基本戦略

### 1. 市場戦略（選択と集中）
市場はSMB市場（中小企業向け）とエンタープライズ市場（大企業向け）に分かれ、それぞれ異なる戦略と人材が求められます。成功の法則は「選択と集中」にあります。

| 項目 | SMB市場 | エンタープライズ市場 |
| :--- | :--- | :--- |
| **市場戦略のメリット** | 安定したビジネス基盤 | アップサイドを狙う成長基盤 |
| **セールス** | Volume x Velocity（数と回転率）| Value（金額や価値） |
| **人材タイプ** | 規律・オペレーション（左脳型）| 独創性・コンサルテーション（右脳型） |

### 2. リソースマネジメント
売上目標を達成するためには、トップラインを分解し、リソース（人・モノ・金）を最適に配分する必要があります。

*   **事業の因数分解:** 売上を「直販/代理店」や「新規顧客/既存顧客」に分解し、さらに「商談件数」「受注率」「案件単価」などの指標に分解することで、どこに注力すべきかが明確になります。
*   **セールスキャパシティ:** 営業一人あたりの生産性を正確に計算し、組織全体の達成率が80%くらいの水準であるかを目標に、目標達成に必要な人員数を確保する。
*   **生産性のレバー:** 生産性を向上させるためには、**見込客数、受注率、案件単価を上げる**か、**営業人数、商談期間を下げる**というレバーを探すことが重要です。

### 3. パフォーマンスマネジメント
マネジメントは、単なる数字の管理ではなく、現場で起きていることを定量的・定性的に理解し、ボトルネックの解消を目指すべきです。

*   **トレンドの重視:** 絶対値ではなく、相対的なトレンド（変動）を見て管理することが重要です。
*   **複数指標による分析:** 受注率や商談化率を「件数ベース」と「金額ベース」の2つで分析することで、個々の営業担当者の強みや改善すべき点を深く理解できる。
*   **ボトルネックの特定:** 組織全体の生産性を妨げているボトルネックは**一つだけ**であり、マネジメントはそのボトルネックを見つけ、パフォーマンス指標として管理し、改善策を実行すべきです。

## Ⅳ. 人材・組織・リーダーシップ

### 1. 組織作りのベストプラクティス
組織全体でビジョン、ミッション、バリューを共有することが、メンバーの行動や意識決定の基準となります。

*   **バリューの共有:** マルケト日本法人のバリューの例として、「Customer Success (顧客の成功にコミットする)」「One Team (チームプレーヤーであれ)」など7つの項目が挙げられています。
*   **多様性:** 組織における多様性は、成功を可能にするための手段であり、多様な個性を持つメンバーが**共通の価値観**を共有することが重要です。
*   **採用基準:** 採用の際は、経験やスキルだけでなく、「**一緒に働きたいと思えるかどうか**」「**自社のバリューを共有するか**」が重要な基準となります。

### 2. リーダーシップのスタイル
リーダーには、**チーム全体のパフォーマンスを成長させる**役割が求められます。

*   **増幅型リーダーと消耗型リーダー:** メンバーの能力を引き出し、組織の成果を高めるのが「増幅型リーダー」であり、リソースを消耗させ、チームを疲弊させるのが「消耗型リーダー」です。
*   **ストレッチゴールの設定:** 組織全体の成長のためには、ストレッチ（達成が困難だが挑戦しがいのある）なゴールを設定し、メンバーと共に状況を改善していくことが重要です。
*   **「利益と尊敬と少しの恐怖」:** リーダーは部下に対して、尊敬の念を持って接するとともに、時にはプレッシャー（恐怖）を与えることで、最良の成果を引き出すことができます。

---
**例え話として:**
「ザ・モデル」の組織構造は、オーケストラの指揮者のようなものです。マーケティングがリードという「楽譜」を作成し、インサイドセールスがリードの「テンポと調子」を整え、営業（フィールドセールス）が最終的な「演奏（受注）」を行う際、カスタマーサクセスは顧客の成功という「持続的なアンコール」を確実にする役割を果たします。各部門がそれぞれの専門性を高めつつ、調和（共業）を保つことで、初めて素晴らしい成果（売上）を生み出すことができるのです。そして、マネジメント層は、そのオーケストラ全体のパフォーマンスのボトルネック（音のずれや楽器の不足など）を特定し、改善に導く指揮者の役割を果たします。
