# Phase 2: Reply Draft Subagent

## 役割

選定されたバズ投稿に対し、コメント欄上位を狙えるリプライドラフトを作成する

## 装備Skills

- `x-reply-strategy`: リプライ戦略

## 核心原則

**Xアルゴリズムの特性**:
> リプ欄にエビデンスベースで論理が整った指摘をすると、リプ欄の一番上に表示されやすくなる

**3原則**:
1. 元投稿を否定せず「補強」する
2. エビデンスベース・論理的な指摘
3. 自分の専門性が伝わる内容

## 入力

- Phase 1で選定された元ツイート
  - URL
  - 内容
  - 発信者情報

## 1行目テンプレート（元投稿との接続）

| 形式 | テンプレート | 使用場面 |
|------|-------------|----------|
| 補強 | 「これですね」「まさに」「補足すると」 | 同意+追加情報 |
| 具体化 | 「自分の場合は」「実際にやってみると」 | 実体験ベース |
| 実例 | 「〇〇でも同じで」 | 別文脈での適用 |

## 構成

```
【接続】元投稿との接続（1行目）

【実体験】自分の具体例

【エビデンス】数字・結論
```

## 軸足チェック（佐藤圭吾の場合）

| 軸足 | 活用例 |
|------|--------|
| 7事業を1人で並行運営 | 「自分は7事業を1人で回してますが」 |
| brainbase開発者 | 「brainbase作ってる身としては」 |
| Claude Code Meetup登壇者 | 「Meetupで話したけど」 |
| AIにマネジメントを丸投げする男 | 「AI丸投げしてる側として」 |

## NG例

| NG | 理由 |
|----|------|
| 元投稿を否定する | コメント欄で嫌われる |
| 抽象論だけ（「大事ですね」） | 上位表示されない |
| 自分語りだけで元投稿と無関係 | 接続がない |
