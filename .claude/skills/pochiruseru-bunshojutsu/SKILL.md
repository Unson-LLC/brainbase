## Triggers

以下の状況で使用：
- 新規リード獲得用のLPや広告文を作るとき
- 読まれない/反応しないコピーを立て直したいとき
- 特徴→欲しい人→ベネフィットの3ステップを素早く回したいとき

この本（『ポチらせる文章術』）に書かれている本質的な事柄は、**「見てもらい、共感してもらい、買ってもらう」**というゴールを達成するためのネット文章術であり、特に**「読まれる提案」**を作成し、商品の販売を最大化するための体系的な手法に集約されています。

以下に、ソースに基づいて抽出された本質的な要素を構造化し、詳細に記述します。

---

## I. 文章作成における基本思想と成果の決定要因

### 1. 成功の80%は書く前に決まっている
広告の成否は、文章を書き始める前の段階でほぼ決まっており、その割合は**80%**に上るとされています。
売れない文章の最大の理由は、**「読まれていない」**ことにあります。どんなに優れた広告やホームページを作成しても、お客様に興味がなければ読まれず、売れない結果となります。

### 2. コピーライティングの本質
コピーライティングの技術は、「文章表現をこねくりまわすこと」ではありません。
重要なのは、**「考える8割」**と**「表現する2割」**であり、読まれる提案を作るために思考を深めることが最も重要です。

### 3. 「読まれる提案」のシンプルなルール
負け組文章（「興味ねー」→「いらねー」→「読まねー」）から抜け出し、「読まれる提案」を作るためのシンプルなルールは、**「欲しい人へ、欲しいものを売る」**ことです。このルールを徹底すれば、他の要素（価格やブランド）に惑わされずに済みます。

---

## II. 新しい顧客を生む「読まれる提案」作成のための3ステップ

新しいお客様を生み出し、売上を向上させる提案は、以下の3つのステップで作成されます。

### ステップ1：商品特徴やメリットの徹底的な洗い出し
まずは、目の前の商品の特徴やメリットを徹底的に書き出します。

### ステップ2：既存ターゲット以外で「欲しい人」を探す
商品が持つ特徴やメリットに、**既存のターゲット層「以外」で**強く反応する「欲しい人」を探します。
視点を変えることで、欲求の高いターゲット層（広告の世界では**「泳ぎ」**と呼ばれる）を発見できる可能性があります。

### ステップ3：強烈に反応するベネフィットを考える
「欲しい人」が、その商品を入手することで得られる**「ハッピーな未来」**、すなわちベネフィットを明確にします。

*   **ベネフィットの定義：** お客様が商品そのものに対してお金を払うのではなく、**その商品を手に入れたことによって解決される問題や、手に入れるハッピーな未来（ベネフィット）に対してお金を払う**という考え方です。
*   ベネフィットは、抽象的な表現ではなく、**具体的**にイメージしやすいように書く必要があります。

## III. 読者の心をつかむ技術：キャッチコピーの作成

キャッチコピーは、広告を読み進めるかどうかを判断する最初の、最も重要な要素です。広告の良し悪しは、キャッチコピーで何より大きく左右されます。

### 1. キャッチコピー作成の10のテクニック
「興味ある」を「読まなきゃ」に変えられるキャッチコピーを作るための10の技法があります。

| テクニック | 概要 | 詳細・効果 | 引用元 |
| :--- | :--- | :--- | :--- |
| **① ベネフィットを語る** | お客様が得るハッピーな未来を伝える。 | 広告表示にベネフィットを載せることは必須。 | |
| **② 読み手を絞り込む** | ターゲットを絞り、具体的な読者に語りかける。 | ターゲットを具体的に絞り込むことで、読者の心に響く。 | |
| **③ 驚きを読ませる** | 読者に意外な発見や驚きを提供する。 | 未完成の情報を提示することで、好奇心を刺激し読み手の心を引きつける（ザイガニック効果）。 | |
| **④ 魅力的なオファーを語る** | 無料のトライアル、保証、特典、サポートなど、お客様が今すぐ契約できる条件を提示する。 | オファーには価格、特典、保証、サポートなどを含める。 | |
| **⑤ 禁止する** | 敢えて行動を禁止し、興味を引きつける（カリギュラ効果）。 | 読み手の本能に訴えかけ、意図的に抑制することで注意を引く。 | |
| **⑥ 数字を上手に使う** | 具体的な数字や単位を用いることで、信頼性を高める。 | 数字があると説得力が増し、貯金や成果などをコントロールして表現できる。 | |
| **⑦ 証拠を見せる** | お客様の声や、専門家・機関からの推薦などの社会的証明を提示する。 | 権威性、信頼性を高め、多くの人から認められている事実を示す。 | |
| **⑧ 結果を見せる** | ビフォーアフターで劇的な変化を視覚的に提示する。 | 写真と共に、変化を数字で表現すると効果が最大化する。 | |
| **⑨ 五感に訴える** | 目、耳、鼻、舌、肌など五感に訴える表現を使う。 | ベネフィットが具体的に伝わり、快楽に満ちた瞬間をイメージさせる。 | |
| **⑩ ストーリーを語る** | 「日常」→「どん底」→「成功」のV字型の構造で、読み手の感情移入を誘う。 | エモさを感じさせ、難解な内容も記憶に残りやすく、分かりやすくなる。 | |

### 2. 効果的なキャッチコピーのテンプレート
提案が分かりにくいキャッチコピーにならないように、以下の5つの効果的なテンプレートを活用できます。

| テンプレート | 名称 | 構成要素 | 該当例 | 引用元 |
| :--- | :--- | :--- | :--- | :--- |
| **①** | **「意外性の高い提案」** | 意外性の高い要素 + 具体的なベネフィット + する方法 | 10年悩んだ原因不明の腰痛を、たった3日で劇的改善した具体的な方法 | |
| **②** | **「ストーリー性が強い提案」** | 不利な状況 + 具体的な成功 + できる | 学年ビリの国立大学に現役合格できたのか？ | |
| **③** | **「強力なベネフィットを約束する提案」** | 強烈なベネフィット + 約束 | 3ヶ月でマイナス10キロ痩せなければ、全額返金します | |
| **④** | **「新規見込み客への広告」** | ターゲットの具体的な状況 + 悩み + あなたへ | イヤイヤ期の子供を前向きのママへ | |
| **⑤** | **「不安をあおる提案」** | 〇〇でこんな間違いをしていませんか？ | 難関個別ダイエットは、実はリバウンドするような間違いをしていませんか？ | |

## IV. ボディコピーの書き方と構成要素

ボディコピーはキャッチコピー以外のすべての文章を指し、長文を書かなくても売れる方法が存在します。

### 1. 売れるボディコピーの構成（6つのステップ）
セールストークを文章に落とし込むことで、短時間で読ませるボディコピーを完成できます。これは訪問販売の営業トークに例えられます。

1.  **煽り効かせる（キャッチコピー）**：最初にお客様の注意を惹きつける。
2.  **近づいてもらう（リードコピー）**：ターゲットの悩みや現状に寄り添い、親近感を持ってもらう。
3.  **信用できる証拠を見せる**：証拠を提示し、商品やあなた自身への信頼性を獲得する。
4.  **ベネフィットを伝える**：ハッピーな未来を具体的に伝える。
5.  **ベネフィットが叶う理由として商品を伝える**：なぜそのベネフィットが実現するのか、具体的な商品（メリット）を説明する。
6.  **今すぐ動いてもらう理由や条件を伝える**：申し込みを促すための具体的な理由や、今動かないと損をする条件を提示する。

### 2. スプリットテスト（A/Bテスト）の活用
キャッチコピーやボディコピーを変更した際、実際に効果が出るかを測定するためにスプリットテスト（A/Bテスト）を行います。スプリットテストは、ネット広告においてレスポンス率の改善、つまり売上を倍増・3倍増させるために非常に重要です。

*   一度に多くのキャッチコピーをテストするのではなく、**一度に1ヶ月だけ**、特定の条件を変えて比較することが推奨されます。
*   初心者は、まずはキャッチコピーを変更し、スプリットテストを行うことが推奨されます。

## V. ネット媒体ごとの戦略と商品認知ステージの応用

ネット媒体ごとにコピーのツボが異なります。効果を倍増させるためには、**商品認知ステージ**を理解し、ターゲットの状況に合わせたコピーを書くことが極めて重要です。

### 1. 商品認知ステージの4段階

ターゲット顧客の「商品に対する認知度と欲求の高さ」に基づいて、以下の4つのステージに分類されます。

| ステージ | 状態 | ターゲットイメージ | 欲求レベル | 引用元 |
| :--- | :--- | :--- | :--- | :--- |
| **① その商品が欲しい** | めっちゃ興味を持っている。 | 商品名や関連キーワードで検索している顧客。購入意欲が最も高い。 | **最も高い** | |
| **② その商品を少し知っているけど、まだ欲しくはない** | 興味はあるが、商品知識は不完全。 | 商品の良さは知っているが、購入を迷っている段階。情報収集をしている。 | 高い | |
| **③ ベネフィットに興味あるもそれを満たす商品を知らない** | 悩みや欲求は持っているが、解決策となる商品自体を知らない。 | 漠然とした悩みや関心を持っているグレーゾーンの顧客。コンテンツマーケティングの対象。 | 中程度 | |
| **④ まったく無関心** | 興味なし、欲しくもない。 | 商品やベネフィットに興味がない。この層へのアプローチは非常に難しい。 | 低い | |

### 2. 媒体ごとのアプローチと商品認知ステージ

| 媒体 | 主なターゲットステージ | アプローチのポイント | 引用元 |
| :--- | :--- | :--- | :--- |
| **リスティング広告** | ステージ①、②、③ | キーワード設定により、複数のステージにアプローチできる。 | |
| **LP (ランディングページ)** | ステージ②のターゲットが多い | LPは「最強のセールスマン」として、商品の魅力を徹底的に説明する。 | |
| **Facebook広告** | 基本的にステージ③ | セグメント（属性）でターゲットを絞り込み、悩みに直結する広告を出す。画像とコピーの両方が重要。 | |
| **リマーケティング** | ステージ③ (リマケ経由でLPへ) | 一度ウェブサイトにアクセスしたが購入に至らなかったターゲット層に、再びLPを配信する。 | |
| **類似オーディエンス** | ステージ③ | 特定のWEB媒体にアクセスした人に近いタイプのユーザーを自動的に探すシステム。欲求レベルの高い人を探せる。 | |
| **メルマガ** | 既存顧客や見込み客 | 件名に「ある要素」を入れることで開封率が向上する。 | |

### 3. LPの構成要素
LP（ランディングページ）は、セールストークの全てを網羅した構成となっており、以下の3つの要素が基本となります。

1.  **大前提：** 1つの商品や1つの提案だけを徹底的に解説する。
2.  **キャッチコピー：** 最初に必ず必要。
3.  **ボディコピー：** ステップに沿って商品を詳細に説明する。
4.  **申し込みフォーム：** 最後にお客様がすぐに申し込めるように配置する。

### まとめとなる比喩
この文章術は、まるで優れたセールスマンが、お客様の購買に至るまでの感情の道のり（興味、欲求、信頼、購入のステップ）を先回りして設計し、適切なタイミングで最も響く言葉（キャッチコピーやベネフィット）を提供し続けるようなものです。**「欲しい人」を正確に見つけ出し、彼らが望む「ハッピーな未来」を具体的な言葉で「約束する」**ことで、インターネット上での販売という困難なタスクを成功に導きます。
