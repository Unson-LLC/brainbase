# Remotion Quickstart

## Typical Commands

Prefer project-local scripts if they already exist. Otherwise use the official Remotion CLI flow for the package manager in the repo.

Common tasks:

- create a starter project
- run local preview
- render a composition
- pass props or input data into the render

## First Deliverable

Aim for this minimum output:

1. One composition
2. One editable headline
3. One brand color prop
4. One successful local render

Anything beyond that is second pass work.
