# Tension Gate（意外性・矛盾・不安の注入）

目的: 量産でも「驚き・揺れ・体温」が出るようにする

## 前提
- 投稿は必ず tension_tag を持つ
  - surprise | contradiction | anxiety
- 本文にタグの内容が実際に反映されていること

## 定義

### surprise（意外性）
期待と結果のズレを入れる
- 例: 減らしたら増えた、止めたら動いた、短くしたら刺さった

### contradiction（矛盾）
両立しないように見える2つの事実を同居させる
- 例: 自動化したら人手が増えた、正論を捨てたら数字が上がった

### anxiety（不安）
締切、失敗、沈黙、未確定を置く
- 例: 返信0、締切14時、レビュー9本、深夜2時

## 仕組み（最小）
1. Draftに tension_tag を付ける
2. 1行目か2行目に「ズレ/矛盾/不安」の核を置く
3. 結果は「驚き」または「実測」まで書く

## NG
- tension_tagだけで本文に反映がない
- 抽象語だけで具体がない
- 教訓で綺麗にまとめすぎる

## 例（短く）

surprise
```
1日30本をやめたら返信が増えた
夜3本は0
朝1本は7
```

contradiction
```
自動化したらレビューが増えた
楽になるはずが詰まりが増えた
```

anxiety
```
締切14時でレビュー9本
朝から手が震えた
```
