# SNS運用ガイド（brainbase標準）

## フォルダ構成
- `README.md` … 本ガイド
- `x_account_profile.md` … 各アカウントの現行プロフィール
- `sns_strategy_os.md` … 戦略メモ（事業OSエンジニア版）
- `templates/` … テンプレ置き場
- `drafts/` … 投稿下書き（スケルトン作成済）
- `log/` … 週次ログ（スケルトン作成済）
- `rules.md` … ガードレール
- `changelog.md` … プロフ・固定ポスト変更履歴
- `reuse.md` … 再利用メモ
- `metrics.md` … 追う指標とダッシュボード場所

## 運用の型
- 毎日: COREドラフトを1本ストック
- 週1: MAGNETドラフトを公開（OSパーツ配布）
- 週次: log に実績と学び、次週計画を記入
- 禁止: ツール感想だけ／一般論だけ → 必ず「OSのどこが変わるか」を1行入れる

## テンプレの使い方
- drafts/slug.md をコピーして下書きを作る
- log/week-YYYY-MM-DD.md に週次実績を記入
- rules/changelog/reuse/metrics は適宜更新
