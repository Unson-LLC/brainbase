採択≠入金だった

補助金って、実は採択決定した時点で支払い義務が発生する

入金後に経費精算だと思ってたら大間違い

採択決定時に「これはお前がやるべき事業」って国のお墨付きが出た瞬間、その事業の費用を立て替える義務が生まれる

だから多くの経営者が後手に回る

俺は金融機関から融資を引いて、そこで先に資金を作る

たとえば、ある事業で採択決定を受けた後、給与計上や設備購入といった経費が発生するけど、実際に入金されるまで100日以上かかることもある

その間、従業員には給料を払わないといけないし、仕入先には支払期限がある

最初は「採択決定されたら入金までの時間」を甘く見ていた

そしたら資金ショートで事業が一瞬止まりかけた

そこからは採択→融資→事業開始→審査→実際入金という4つのフェーズごとに、どこでどの資金が足りなくなるかを徹底的に可視化するようにした

4つの事業を同時に回してるから、各フェーズのリスクが見えないと経営判断ができないどころか詰む

結局、補助金採択→銀行融資→事業開始→入金決定審査→実際入金という4段階の資金フローを可視化し、各フェーズでの資金調達リスクを事業計画に組み込む手法。
