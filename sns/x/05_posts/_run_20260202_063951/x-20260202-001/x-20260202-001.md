税理士23事務所への価格説明で、メール+資料送付っていう『きちんとした』やり方をしてたんだけど、誤解が増える一方だった

で、思い切って説明会で顔見せして業務ツールで個別対応に変えたら、連絡漏れゼロになった

それまでXでコミュニティ作ったら「見てない」って言われて、結局チャットワークに移したんよ

デジタルネイティブじゃない層には、新しいツール=新しい認識齟齬の源泉なんだと気づいた

1人で4事業回してると、SNSの到達率より『相手がいつも使ってる場所に合わせる』ほうが、結果的に速いことがわかる

情報は完璧でも、相手の日常にない場所にあっては意味がない

既存の業務ツールを開く習慣のほうが、新しい通知より圧倒的に強い
