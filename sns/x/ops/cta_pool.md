# CTA Pool (X)

目的: CTAの単調化を避け、反応・導線・会話のバリエーションを作る

## カテゴリ

### コメント誘導
- コメントで教えて
- 一言で教えて
- 経験ある？コメントで
- どっち派？
- ここ詰まってる人いる？
- あなたのやり方、教えて

### 反応誘導（軽いアクション）
- 保存して後で見返して
- メモ代わりに保存
- 刺さったらメモに

### 導線誘導（プロフィール/固定ポスト/note）
- 続きはプロフィールの固定ポストへ
- 固定ポストにまとめた
- 詳細はnoteに書いた（プロフィールから）
- 具体手順は固定ポストに置いた
- プロフィールから読める

### 共有誘導（穏やか）
- 同じ悩みの人に教えて
- 仲間がいたら一言だけでも

## 運用ルール
- 疑問形は全体の40%程度に抑える
- 導線誘導は全体の20〜30%に抑える
- 同一CTAの連続は禁止
- 連続でプロフィール誘導は避ける
