### 4. 筆者の再解釈【800文字】

```
## 僕なりの「翻訳」：現場で使える3スキル

元記事の3スキルを、日本の現場で使える形に翻訳してみた。

| 元記事 | 筆者の翻訳 | 現場での意味 |
|--------|-----------|--------------|
| 志を抱く能力 | **課題設定力** | 「何を問うか」を決める力 |
| 判断力 | **志を抱く能力** | プロジェクトを牽引する意思力 |
| 真の創造性 | **価値設定能力** | 「何に価値があるか」を決める力 |

### なぜ「翻訳」したのか

元記事の3スキルは、経営層向けの抽象的な表現だ。
でも、現場のマネージャーには「明日から何をすればいいか」が見えない。

僕が翻訳した3スキルは、より具体的なアクションに落とせる：

**1. 課題設定力 = 問いを立てる能力**
AIは答えを出すのは得意だが、問いを立てることはできない。
「売上を上げるには？」という問いの設定自体が、人間の仕事。

**2. 志を抱く能力 = プロジェクトを牽引する意思力**
AIは指示に従うが、「なぜやるのか」を語ることはできない。
チームを動かすのは、リーダーの「志」だ。

**3. 価値設定能力 = 何に価値があるかを決める力**
AIは効率化は得意だが、「何が大切か」は決められない。
顧客にとっての価値、社会にとっての価値を定義するのは人間だけ。
```

**ポイント**:
- 表形式で元記事との対比を可視化
- 各スキルに「現場での意味」を追加
- 読者が「明日から実践できる」具体性

### 5. 各スキルの深掘り【600文字】

```
## 3スキルを日常業務に落とし込む

### 課題設定力を鍛えるには

❌ AIに「何をすべきか」を聞く
✅ まず自分で「何が問題か」を定義し、AIにはその解決策を聞く

例：
「売上が伸びない原因は？」（❌ AIに問いを委ねている）
「新規顧客の獲得コストが高い。これを下げる方法は？」（✅ 問いを自分で設定）

### 志を抱く能力を発揮するには

❌ 「AIが言ってるから正しい」で説明する
✅ 「私がやりたいからやる」で説明する

例：
「ChatGPTによると、この施策が有効らしいです」（❌ 他人事）
「私はこの施策が正しいと信じています。理由は...」（✅ 自分ごと）

### 価値設定能力を使うには

❌ AIの提案をそのまま採用
✅ AIの提案を「うちの価値観」でフィルタリング

例：
AIが「コスト削減のため人員削減を」と提案 → 自社の価値観で判断
「うちは"人を大切にする"会社だから、別の方法を探す」
```

**ポイント**:
- 各スキルに「悪い例」「良い例」を提示
- 読者（田中裕也）の日常業務に直結する例
