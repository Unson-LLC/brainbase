【フック】
AIに全部任せるやつ、マジでやばい

【本文】
「AIで自動化！」って言ってる人、多すぎる

でも俺の経験だと、**完全自動化は幻想**

理由：
Claude Code の Team 機能で3つのSubagent並列実行したけど、
1つのAgentに全部任せた時は7時間かかってた

3つに分けたら40分

つまり、「AIに任せる」じゃなく「AIをどう分けるか」が勝負

完全自動化を目指すより、**半自動化で人間が判断する設計**のほうが速い

【締め】
AIは道具
使い方を間違えたら、逆に遅くなる
