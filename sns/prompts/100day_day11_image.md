# Phase 3 Image Prompt

## 判断

- **投稿タイプ**: ストーリー型（失敗談・学び）
- **選択テンプレート**: `incident`
- **選択理由**:
  - フックは「娘との夕食の時間を死守したい」という強い感情トリガー
  - 「パパまだ？」という娘の言葉が原体験
  - 解決策（GAS、Slack通知）ではなく、**問題の切迫感**を視覚化すべき
  - `incident`は「衝撃・警告」で感情トリガーを引く。スクロール停止させる効果
  - `confession`は「内省的・静か」だが、今回は「死守したい」という強い意志があるので`incident`がより適切

## 画像プロンプト（日本語）

```
日本語のincidentビジュアルを作成してください：

テーマ: 娘との夕食の時間を死守したい

デザイン要件:
- 中央に時計（17:45を指す）と警告マーク
- 左側：オフィスで働く人物シルエット（焦りの表現）
- 右側：食卓で待つ子供のシルエット（期待の表現）
- 吹き出し：「パパまだ？」
- 緊張感のある色使い（赤・黒・オレンジ）
- 「デッドライン」「17:45」の文字を強調
- 解決策（GAS、Slack、自動化）は絶対に含めない
- 横長レイアウト（16:9アスペクト比）

重要: すべて日本語で表示してください。
解決策は隠し、問題の切迫感のみを視覚化してください。
```
