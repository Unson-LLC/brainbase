# Phase 3 Image Prompt

## 判断

- **投稿タイプ**: ストーリー型（失敗告白・矛盾の気づき・変化）
- **選択テンプレート**: `confession`（告白/懺悔）
- **選択理由**:
  - このドラフトは「家族のために働いてるのに、家族との時間がない」という矛盾への気づきと告白がコア
  - 派手な事件風（incident）ではなく、静かで内省的な雰囲気が投稿トーンに合致
  - 「暴露するけど」で始まる告白形式
  - 解決策（AIに丸投げ）は画像では絶対に見せない

## 画像プロンプト（日本語）

```
日本語の「失敗告白」ビジュアルを作成してください：

テーマ: 家族のために働いてるのに、家族との時間がない

告白内容:
- 娘が「パパ、今日は遊べる？」って聞いてきた
- 「ちょっと待って」って言いたくなかった
- 気づいたら21時、娘はもう寝てる

デザイン要件:
- 薄暗い背景（ダークブルーまたはダークグレー）
- 上からのスポットライトが当たっている（告白の「場」感）
- 大きな日本語テキストでテーマを表示：「家族のために働いてるのに、家族との時間がない」
- 派手な色は使わない（地味だが重い雰囲気）
- 時計が21:00を指しているシルエット、または暗いオフィスの窓
- 子供の寝顔のシルエット（遠くに、小さく）
- 「...」や沈黙を感じさせる余白
- 横長レイアウト（16:9アスペクト比）

重要な制約（オチネタバレ防止）:
- AIや自動化の要素は一切含めない
- mana-weekly、エージェント、効率化の言葉は一切含めない
- 解決策や「18時に帰れるようになった」結果は一切示さない
- 問題提起のみ（矛盾への気づきだけ）
- 視聴者が「どうやって解決したの？」と続きを読みたくなる状態を作る

スタイル: 内省的、静か、当事者視点。ニュース速報のような派手さではなく、一人で振り返っている瞬間を切り取った雰囲気。

重要: すべてのラベル、タイトル、説明文は日本語で書いてください。英語は一切使わないでください。
```

## 適用した画像生成テクニック

| テクニック | 適用理由 |
|-----------|---------|
| **感情の視覚化** | 「21時」「娘の寝顔」という具体的な感情トリガーをシルエットで表現。解決策ではなく感情を伝える |
| **オチ隠し** | AI・自動化・効率化の要素を一切含めず、問題提起のみ。「続きが気になる」状態を作る |
| **confession テンプレート活用** | 派手な警告ではなく、静かで内省的な告白の雰囲気。ドラフトのトーンに合致 |
| **時間の可視化** | 「21:00」の時計シルエットで、仕事が終わらない感覚を直感的に伝える |
| **余白の活用** | 「...」の沈黙感を余白で表現。情報を詰め込まず、感情を残す |
