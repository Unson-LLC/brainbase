# 原則：感情を動かせ

**X伸びてない人が間違えてること = 情報の垂れ流し**

いいね・フォロー・リポストは、情報の正しさでは動かない。**感情が動いたとき**に初めて行動が起きる。

| 感情 | 反応 |
|------|------|
| すご！ | 保存・リポスト |
| がんばろ！ | いいね・フォロー |
| イラッ！ | 引用RT・コメント |
| ありがとぉ！ | フォロー・DM |
| もっと知りたい！ | プロフ見る・フォロー |

**だから**: フックで感情を煽る → 本文で納得させる → CTAで行動させる

# ガードレール
- 禁止: ツール感想だけ、一般論だけ、事業OSと無関係なAIニュース
- 必須: 「OSのどこが変わるか」を最低1行、問いかけCTAで締める
- CTA: 「〜な人いる？」「みんなはどうしてる？」など、コメントしやすい問いで締める（DM誘導は避ける）
- トーン: 簡潔・具体・再現性。誇張NG、数字は根拠付き

## 優等生発信NG（無難な発信は無関心を生む）

**原則**: 売上5億円前後までの経営者で、まだ世の中にほとんど知られていない段階なら、SNSで「優等生」をやっていても、正直、誰の記憶にも残らない。

**無難な発信が生むのは、共感ではなく「無関心」**

嫌われる可能性を引き受けて発信して、はじめて「それでもこの人が好きだ」という人が現れる。その"好き"が、時間をかけて信頼に変わり、やがて仕事やご縁につながっていく。

### NG表現（優等生発信）
- ❌ 「〜が重要です」「〜をおすすめします」（教科書的）
- ❌ 「〜という考え方もあります」（両論併記、優柔不断）
- ❌ 「〜だと思います」（自信がない印象）
- ❌ 「いろいろな意見があると思いますが」（波風を立てない）

**理由**: 安全運転だけで人の心を動かし、結果を出してきた経営者は、まだ見たことがない。

**例外**:
- 技術的な説明で、両論併記が必要な場合（比較表等）
- 専門性が求められるトピック（セキュリティ、法的リスク等）

### OK表現（覚悟を示す）

**原則**: 軸足の範囲内（x_account_profile.md の「専門領域」）でのみ、以下の表現を使用可

- ✅ 「〜は間違ってる」（断定）
- ✅ 「〜はやめた方がいい」（警告）
- ✅ 「俺は〜した」（実体験、リスク引き受け）
- ✅ 「〜な経営者を、何人も見てきた」（経験による断言）
- ✅ 「〜って本当に正しいのか？」（常識への疑問提起）
- ✅ 「マジで危険だと思ってる」（熱量、覚悟）

**重要**: 軸足から外れた状態でのエッジの効いた発言は、炎上リスク + 信頼失墜を招く。必ず x_account_profile.md の「軸足」セクションを確認すること。

# imp伸ばしの構造（1万imp狙い）

## 基本構造
```
【フック】← 常識破壊。ここで「続きを読む」を押させる
（続きを読む）
【本文】← 悩み→判断→結果。ブクマ・リスト獲得に影響
【CTA】← 問いかけで締める
```

## フックの鉄則
- **長文にする**（続きを読むが出る長さ）
- **常識破壊を1行目に入れる**（「本当は内緒だけど」「暴露するけど」「実は〜」等）
- フックの内容は正味なんでもいい。押させることが目的
- 1万impまでは中身適当でも出る。3万imp〜狙うなら本文の質も重要

## フックパターン集

### A. 常識破壊・秘密系（最も効く）
- 「本当は内緒にしときたいんだけど、〜」
- 「暴露するけど、〜」
- 「実は〜してる」
- 「〜はやめた方がいい。理由は〜」
- 「〜って言われるけど、俺は逆で〜」
- 「みんな〜してるけど、それ間違ってる」
- 「〜年やってわかったけど、〜」

### B. 禁止・警告系（カリギュラ効果）
- 「〜な人は読まないでください」
- 「〜してる人、マジでやめた方がいい」
- 「〜だけは絶対やるな」

### C. 意外性・驚き系（ザイガニック効果）
- 「〜したら、〇〇になった」（意外な結果）
- 「〜だと思ってたけど、実は〜だった」
- 「〇〇日で〜できた方法」

### D. 数字・実績系
- 「〇〇%の人が知らない〜」
- 「〇〇日で〜した」
- 「たった〇〇で〜」

### E. 問いかけ・不安喚起系
- 「〜でこんな間違いしてませんか？」
- 「なぜ〜なのか？」
- 「〜してる人、他にいる？」

### F. ストーリー系（V字型）
- 「〜で失敗した→〜で逆転した」
- 「〜できなかった俺が、〜できるようになった理由」

### G. 簡単・即効系
- 「〜するだけで〜」
- 「誰でもできる〜」
- 「3分でわかる〜」

## 投稿後の行動
- **投稿後3分はアクティブを保つ**（リプ返、いいね等）
- これだけで伸びが変わる
