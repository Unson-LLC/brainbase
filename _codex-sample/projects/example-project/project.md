## 🎯 プロジェクト概要

Example Projectは、brainbaseのサンプルデータとして作成された架空のプロジェクトです。実際のプロジェクトでは、以下の情報を記載します:

- プロジェクトの目的

## 🚀 戦略（01_strategy）

### ビジョン

brainbaseを使って、すべてのプロジェクトを効率的に管理する。

### ミッション

- プロジェクト情報を一元管理
- タスクの可視化と自動化
- チーム間のコミュニケーション円滑化

### ICP（理想顧客プロファイル）

**企業規模**: スタートアップ〜中小企業（社員数10-100名）

**課題**:
- プロジェクト管理が属人化している
- タスクの進捗が見えない
- 情報が分散している

**ニーズ**:
- シンプルで使いやすいPMツール
- 自動化による工数削減
- AIによるサポート

## 📊 KPI（02_kpi）

### 主要KPI

| KPI | 目標値 | 測定頻度 | 現在値 |
|-----|--------|----------|--------|
| アクティブユーザー数 | 100名 | 週次 | 0名 |
| タスク完了率 | 80% | 週次 | - |
| 平均タスク処理時間 | 3日以内 | 月次 | - |
| ユーザー満足度 | 4.0/5.0 | 四半期 | - |

### 測定方法

- **アクティブユーザー数**: 週1回以上ログインしたユーザー
- **タスク完了率**: 期限内に完了したタスク数 ÷ 総タスク数
- **平均タスク処理時間**: タスク作成から完了までの平均日数
- **ユーザー満足度**: 四半期アンケート（5段階評価）
